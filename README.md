# PromSys_SunPower

SunPower is a dynamic web development project created for PromSys. It features a captivating landing page with smooth animations, enhanced visuals, and a seamless user experience. The HTML and CSS are optimized for responsiveness across devices, ensuring a consistent presentation. With advanced features like parallax effects on images, SunPower sets a new standard for engaging web design.
